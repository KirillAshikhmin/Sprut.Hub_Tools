# Sprut.Hub_Tools
Сценарии и шаблоны для Sprut.Hub

Коллекция полезных логических и глобальных сценариев для автоматизации умного дома на платформе Sprut.Hub.

---

## 📋 Сценарии

### 🏠 Управление освещением
- [Циркадное освещение](https://github.com/KirillAshikhmin/Sprut.Hub_Tools/tree/main/CircadianLight "Циркадное освещение") — автоматическое изменение цвета и яркости ламп в зависимости от времени суток
- [Мастер-выключатель](https://github.com/KirillAshikhmin/Sprut.Hub_Tools/tree/main/TurnOffAllLight "Мастер-выключатель") — выключение всех ламп одной кнопкой
- [Адаптер минимальной яркости ламп](https://github.com/KirillAshikhmin/Sprut.Hub_Tools/tree/main/MinBrightAdapter "Адаптер минимальной яркости ламп") — автоматическая установка минимальной яркости для ламп

### 🌡️ Климат и температура
- [🌡️ Виртуальный термостат](https://github.com/KirillAshikhmin/Sprut.Hub_Tools/tree/main/VirtualThermostat "🌡️ Виртуальный термостат") — создание полнофункционального термостата из датчика температуры и реле
- [💧 Виртуальный гигростат](https://github.com/KirillAshikhmin/Sprut.Hub_Tools/tree/main/VirtualHygrostat "💧 Виртуальный гигростат") — создание полнофункционального гигростата из датчика влажности и реле
- [Реле для термостата](https://github.com/KirillAshikhmin/Sprut.Hub_Tools/tree/main/ThermostatRelay "Реле для термостата") — управление реле на основе состояния термостата
- [🌀 Управление реле по скорости вентилятора](https://github.com/KirillAshikhmin/Sprut.Hub_Tools/tree/main/FanSpeedRelayControl "🌀 Управление реле по скорости вентилятора") — управление реле в зависимости от скорости вентилятора термостата
- [Внешний датчик температуры для термоголовки Aqara E1](https://github.com/KirillAshikhmin/Sprut.Hub_Tools/tree/main/ExternalTempSensor "Внешний датчик температуры для термоголовки Aqara E1") — использование внешнего датчика температуры для термоголовки
- [Средняя температура и влажность в комнате](https://github.com/KirillAshikhmin/Sprut.Hub_Tools/tree/main/AverrageTempAndHum "Средняя температура и влажность в комнате") — вычисление средних значений температуры и влажности

### 🪟 Управление окнами и шторами
- [🪟 Виртуальное окно/штора](https://github.com/KirillAshikhmin/Sprut.Hub_Tools/tree/main/VirtualWindow "🪟 Виртуальное окно/штора") — управление окном или шторой с двунаправленным мотором через два реле

### ⏰ Планирование и таймеры
- [📅 Планировщик расширенный](https://github.com/KirillAshikhmin/Sprut.Hub_Tools/tree/main/Scheduler "📅 Планировщик расширенный") — расширенный планировщик с поддержкой дней месяца, месяцев и множественных времен
- [⏰ Параметр-Таймер](https://github.com/KirillAshikhmin/Sprut.Hub_Tools/tree/main/ParameterTimer "⏰ Параметр-Таймер") — настраиваемая задержка с удобным управлением через интерфейс
- [🌅 День-ночь](https://github.com/KirillAshikhmin/Sprut.Hub_Tools/tree/main/DayNight "🌅 День-ночь") — автоматическое включение и выключение выключателя на основе времени восхода и заката солнца
- [📅 IsDayOff - Выходной ли день?](https://github.com/KirillAshikhmin/Sprut.Hub_Tools/tree/main/IsDayOff "📅 IsDayOff - Выходной ли день?") — проверка выходных дней через интернет-сервис isdayoff.ru
- [❄️☀️ Зима/Лето](https://github.com/KirillAshikhmin/Sprut.Hub_Tools/tree/main/WinterSummer "❄️☀️ Зима/Лето") — автоматическое управление выключателем в зависимости от сезона года

### 🎛️ Управление устройствами
- [📈 Пошаговое изменение характеристики по кнопке или датчику касания](https://github.com/KirillAshikhmin/Sprut.Hub_Tools/tree/main/StepChange "📈 Пошаговое изменение характеристики по кнопке или датчику касания") — универсальный сценарий для управления любыми характеристиками устройств через кнопку или датчик касания
- [👁️ Скрыть устройство если выключено](https://github.com/KirillAshikhmin/Sprut.Hub_Tools/tree/main/HideIfTurnOn "👁️ Скрыть устройство если выключено") — автоматическое скрытие выключенных устройств в интерфейсе

### 🔋 Мониторинг и диагностика
- [🔋 Отслеживание батарей умных устройств и сводка по ним](https://github.com/KirillAshikhmin/Sprut.Hub_Tools/tree/main/Battery "🔋 Отслеживание батарей умных устройств и сводка по ним") — универсальная система мониторинга батарей для всех умных устройств
